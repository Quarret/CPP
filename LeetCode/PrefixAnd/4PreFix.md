# 四种区间的前缀和写法
## 前缀数组的意义
```
nums = [1, 2, 3, 4, 5]
pre  = [0, 1, 3, 6, 10, 15]

pre[i] = \sum[0, i] (i为该数是数组中的第几个数. i = 0时特别为0) 
```

## [l, r]
```
[l, r] = [0, r] - [0, l - 1]
res = pre[r] - pre[l - 1]
```

## [l, r)
```
[l, r) = [l, r - 1] = [0, r - 1] - [0, l - 1]
res = pre[r - 1] - pre[l - 1]
```

## (l, r]
```
(l, r] = [l + 1, r] = [0, r] - [0, l]
res = pre[r] - pre[l]
```

## (l, r)
```
(l, r) = [l + 1, r - 1] = [0, r - 1] - [0, l + 1]
res = pre[r - 1] - pre[l + 1]
```
